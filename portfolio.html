Project 1 — Meta Ads ROAS (Live)
Small Business · Live Project · Real Data

Meta Ads ROAS Prediction Dashboard

A live end-to-end analytics project built on real Meta/Facebook ad data. A Random Forest model predicts return on ad spend per campaign — actual vs predicted ROAS shown side by side in a live Metabase dashboard connected to PostgreSQL.

Python scikit-learn Random Forest pandas PostgreSQL psycopg2 Metabase
Data sourceReal Meta Ads Manager export
ModelRandomForestRegressor
Targetpurchases_conversion_value
Real Live Meta ad data
29 Campaign features modelled
Live Metabase dashboard

What was built

Real data pipeline
Exported raw Meta Ads Manager CSV (spend, impressions, clicks, CTR, video metrics, conversions) → cleaned in Python → loaded into PostgreSQL via psycopg2.
ROAS prediction model
RandomForestRegressor trained on spend, impressions, CTR, and video engagement metrics to predict purchases conversion value → calculated predicted ROAS per campaign.
Predictions written back to DB
Used psycopg2 execute_batch to UPDATE the PostgreSQL table with predicted values — no CSV export needed. Metabase reads live from the database.
Live Metabase dashboard
Metabase connected directly to PostgreSQL showing actual vs predicted ROAS side by side — a real tool a client's marketing team could use daily.
